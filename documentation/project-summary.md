# Project Summary

## Business Context

Air Cargo is an aviation company that provides passenger and freight transportation services. The company needs SQL-based reporting to better understand passengers, routes, ticket sales, and customer behavior.

## Project Objective

The objective of this project is to analyze the airline database and generate business insights related to customer activity, route performance, ticket revenue, and service eligibility.

## Main SQL Tasks

- Created route details table with constraints
- Queried passengers travelling across selected route ranges
- Calculated business class passenger count and revenue
- Extracted registered customers who booked tickets
- Identified customers travelling with specific airline brands
- Used Group By and Having for customer segmentation
- Applied window functions for class-based price analysis
- Created views for business class customers
- Built stored procedures for route and passenger analysis
- Created a stored function for complimentary service eligibility
- Used cursors to extract customer records

## Key Learnings

This project strengthened my ability to write business-oriented SQL queries, work with relational databases, create reusable database objects, and translate operational requirements into structured SQL analysis.
